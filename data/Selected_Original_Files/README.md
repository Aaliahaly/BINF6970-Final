# Selected Original Files
This folder contains the subset of original TCGA LGG dataset files selected for use in this project. All files are preserved in their original form, with no cleaning, transformation, or preprocessing applied, and serve as the direct input to the data processing pipeline.

## Note
The files `data_mutations.txt` and `data_cna.txt` have been compressed (zipped) to enable upload to this repository.
The file `data_mrna_seq_v2_rsem.txt` is not included due to its large size. It should be downloaded directly from the original data source to fully reproduce the pipeline.
